#  vue和python笔记

##  1. python笔记

### 		1.1 知识要点

- vscode创建python虚拟环境

```  python
1 在当前文件夹新建终端并输入以下命令：
python -m venv venv01    # 
2 在当前目录激活虚拟环境
.\venv01\scripts\activate
3 退出虚拟环境
 在虚拟环境提示符下执行 deactivate

```

- vscode 配置识别python虚拟环境

```python
在settings.json中添加
"python.venvPath": '虚拟环境的绝对路径'   #  如: 'd:\\venv'
```
